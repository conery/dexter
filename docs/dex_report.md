# `dex report`

#### Usage

```
$ dex report --help
usage: dex report [-h] [--start_date D] [--end_date D] [--month M] [--no_budget] [--abbrev] [--grouped] [A ...]

positional arguments:
  A               accounts

options:
  -h, --help      show this help message and exit
  --start_date D  starting date
  --end_date D    ending date
  --month M       define start and end dates based on month name
  --no_budget     remove budget transactions
  --abbrev        print abbreviated names
  --grouped       group by account name
```
